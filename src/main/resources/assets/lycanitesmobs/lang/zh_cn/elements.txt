element.fire.name=火
element.fire.description=一种代表等离子体和原始能量的基本元素，它可以从酸、熔岩、闪电、光和阴影中提取。
#
element.water.name=水
element.water.description=一种代表流体的基本元素，主要是水本身。水可以从复合元素中提取：酸、植物、凛冰、仙灵和剧毒。
#
element.earth.name=大地
element.earth.description=一种代表固体形态的基本元素，从泥土到岩石、晶体甚至金属或骨头。它可以从复合元素中提取：熔岩、植物、相位、重力和震动。
#
element.air.name=风
element.air.description=一种代表气体的基本元素，但通常只因其控制气流的能力而为人所知。它可以从闪电、凛冰、相位、以太和地狱等复合元素中提取。
#
element.order.name=秩序
element.order.description=一种代表序列和平衡的基本元素，虽然这些都是非常假设的概念，但秩序在魔法宇宙中有着非常物理的存在，其基本形式是原始力。秩序可以从复合元素中提取：光、仙灵、重力、以太和虚空。
#
element.chaos.name=混沌
element.chaos.description=一种代表熵、增长和破坏的基本元素。它可以从复合元素中提取：阴影、剧毒、震动、地狱和虚空。
#
element.acid.name=酸
element.acid.description=一种代表腐蚀和物理消耗的复合元素，但可以扩展为广泛的化学性质，而且尽管名称为酸，它也涵盖了基本物质。酸可以由基本元素火和水融合而成，因为它既体现了火的能量特性，又体现了水的流体物质特性。
#
element.lava.name=熔岩
element.lava.description=一种代表具体热量的复合元素，而火更多的是等离子体和能量，而熔岩则是能量储存和热量的影响。熔岩可以从基本元素火和大地融合，因为它体现了火的能量方面和大地的致密质量方面。
#
element.lightning.name=闪电
element.lightning.description=一种代表电流和能量的复合元素。闪电可以从基本元素火和风中融合，因为它体现了火的能量和等离子体方面以及风的流动和不接地方面。
#
element.light.name=光
element.light.description=一种代表辐射的复合元素，使其成为一种非常危险的力，因为它的范围从以该元素命名的可见光谱到可能造成严重损害的更致命的辐射范围。光可以从基本元素火和秩序融合，因为它体现了火的能量方面和秩序的精致。
#
element.shadow.name=阴影
element.shadow.description=一种代表将黑暗为阻止辐射或能量产生的力量的复合元素。阴影可以从火和混沌中融合，因为它体现了火的能量方面，然后被混沌颠倒成一种对抗能量创造的力量，将其消除。
#
element.arbour.name=植物
element.arbour.description=一种代表生长和自然的复合元素，它紧紧围绕着大量利用这种元素的植物。植物可以从水和大地融合，因为它体现了水的流动形式和大地的固体存在。
#
element.frost.name=凛冰
element.frost.description=一种代表寒冷的复合元素，它能有效地排斥任何形式的热。凛冰可以从水和风中融合，因为它体现了水的流体冷却性质和风的冷却流动。
#
element.fae.name=仙灵
element.fae.description=一种代表生命和适应的复合元素。仙灵可以从水和秩序中融合，因为它体现了水的流动性和秩序的结构。
#
element.poison.name=剧毒
element.poison.description=一种代表化学物质的复合元素，通常是有毒或毒液以及麻醉剂和药物，但也可以有许多其他形式，如油。剧毒可以从水和混沌中融合出来，在混沌中，水体现出一种可以被改变的流动形式，这种形式被混沌操纵成各种奇怪的化学物质。
#
element.phase.name=相位
element.phase.description=一种代表移相或精神实相的复合元素，一个分层在物质世界之上的空间，它是精神的构成和使用。相位可以从基础元素大地和风中融合，因为它体现了大地的固体形态和风的流动性非固体性。
#
element.gravity.name=重力
element.gravity.description=一种代表万有引力、磁性、键合、压缩和人造结构的复合元素。重力可以从大地和秩序中融合，在那里它体现了大地的固体物理存在以及秩序的结构和规律。
#
element.quake.name=震动
element.quake.description=一种代表震动、声音、燃烧和破坏的复合元素。地震可以从大地和混沌中融合，在那里大地呈现出固体形态，然后被混沌分裂。
#
element.aether.name=以太
element.aether.description=一种代表纯洁、神性、信仰和崇拜的复合元素。以太可以从风和秩序中融合，因为它用秩序的结构和规律来体现风的流动。
#
element.nether.name=地狱
element.nether.description=一种代表腐败、恶魔、欲望和反叛的复合元素。地狱可以从风和混沌中融合，在那里风体现流动，混沌将其扭曲成腐败。
#
element.void.name=虚空
element.void.description=一种代表虚无和删除的复合元素。秩序体现了结构和一致性，秩序体现了熵和随机性，混沌抵消了秩序和混沌的融合。
